# whitelist-dental.github.io

Live front end for **Whitelist Dental**, a dentist due-diligence app: search a
dentist and get one dossier combining license/board records, disciplinary
actions, malpractice litigation, and reviews — with every adverse finding traced
to its source.

This is a static build of the React app that calls the hosted API at
`https://whitelist-dental.vercel.app`. Source lives in
`blackhawkyama/whitelist-dental` (`frontend/`); rebuild with
`VITE_API_BASE=https://whitelist-dental.vercel.app npm run build` and copy
`dist/` here.

Built with AI assistance (Claude), human-directed.
